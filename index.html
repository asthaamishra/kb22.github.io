I am pursuing Doctor of Philosophy in Computer Science at Rensselaer Polytechnic Institute, Troy, New York.

I have worked on many Machine Learning, Deep Learning, Data Science and Data Analytics projects including Mortality Minder, COVID Minder, Color identification, Malaria Cell Identifier, Activity Recognition, Heart Disease Prediction, Web Scraping and many more.

I have extensive knowledge of Machine Learning algorithms and have worked with Python and R including their numerous libraries such as Tensorflow, Pytorch, Sklearn, Dplyr and Leaflet. I have developed a template for a Machine Learning application using Flask and React.

I'm also great with frontend web application development using HTML, CSS and JavaScript.
